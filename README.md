# Personal Budget Tracker

A very simple web-based personal budget tracker to track your financial income and expense.

## Features

- **Add Transactions**: Record income and expenses with descriptions
- **Show Balance**: View your current balance updated instantly
- **Transaction History**: See all your transactions in a table format with ID, remark, amount, and type
- **Visual amount**: make the color of the amount respondance to whether it is negatice or positive
- **Transaction Counter**: Show the total number of entries


## How to Use

1. **Open the Application**: Open the app.html in your browser
2. **Enter Amount**: Input a positive or negative number (positive for income, negative for expenses)
3. **Add Description**: Write a brief note or remark about the transaction
4. **Click "Add Expense"**: Submit the transaction to add it to your history
5. **View Balance**: Check your current balance and total entries at the top
6. **Review History**: Scroll down to see all your transaction records

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
  - Closures
  - Higher-order functions
  - Arrow functions
  - Spread operator

## Key JavaScript Concepts

- **Closure**: ID generator that maintains state across function calls
- **Higher-Order Functions**: `reduce()`, `forEach()`, `filter()` for data processing
- **Immutability**: Uses spread operator to create new arrays instead of mutating existing ones
